# Virtual Canvas (Prototype)

Fullstack prototype for a dynamic photo canvas with SQL database backend for KTP Project.

Features
- Upload photos via Camera(backend stores images on disk and image data in SQLite database)
- Drag photos around on a large canvas
- Canvas expands dynamically when photos are dragged near edges
- Filter photos by feature key/value using SQL queries

### Frontend
- **HTML**
- **CSS**
- **Javascript**

### Backend

- **sql.js**
